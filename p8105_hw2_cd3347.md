Read and clean the Mr. Trash Wheel sheet:

``` r
mr_trash_tidy = 
  read_excel("./data/202309 Trash Wheel Collection Data.xlsx", sheet = 1,
             range = "A2:N586") %>% 
  janitor::clean_names() %>% 
  mutate(homes_powered = (weight_tons * 500) / 30) %>% 
  mutate(year = as.numeric(year)) %>% 
  mutate(wheel_name = "mr_trash") %>% 
  relocate(wheel_name)
```
